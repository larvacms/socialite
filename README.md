LarvaCMS Alipay Baidu QQ Weibo Wechat WechatWeb 登录接口

## 环境需求

- PHP >= 7.2

## Installation

```bash
composer require larvacms/socialite -vv
```

支持平台：
- wechat
- wechat_web
- qq
- weibo
- baidu
- alipay
